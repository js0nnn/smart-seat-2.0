Current Status

This page tells about the current status of the driver in real time
by interacting with the raspberry pi following which the data is projected
to the frontend with slight modifications.

Drivers log

This page logs information about the drivers activites once the raspberry pi is active
and continues to do so till the pi is turned off ,this feature was implemented so that the
authorized personal can monitor what is going on and refer the data at a later time if required.

Alerts

Should the driver be drowsy or in any condition be asleep despite the features implemented
to prevent it or in a worse case scenario encounter any accident the pi sends out information
regarding the incident which is displayed on this Webpage while also alerting the repective emergency personal
